# Add OpenAlex work IDs to the register table (addresses register#185)

Add OpenAlex work IDs to the register table (addresses register#185)

## Usage

``` r
add_openalex_ids(register_table, register)
```

## Arguments

- register_table:

  The register table

- register:

  The register from the register.csv file

## Value

The register table with added "OpenAlex" column
