# codecheck: Helper Functions for CODECHECK Project

This contains helper functions for codecheckers for conducting
CODECHECKs, e.g., authoring CODECHECK certificates
(<https://codecheck.org.uk>), and the rendering of the CODECHECK
Register (<https://codecheck.org.uk/register/>).

## See also

Useful links:
